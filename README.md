# NextKart
NexCart is a secure and scalable e-commerce application built with .NET MVC (.NET 9). It follows industry-standard architecture with separation of concerns, Repository–Service pattern, and EF Core (Database First). The system includes separate Web and Admin applications with JWT-based authentication and role-based authorization.

Tech Stack (Locked for this project)

Framework: ASP.NET MVC (.NET 9)

Architecture: Layered Architecture

Data Access: EF Core – Database First

Patterns: Repository + Service Pattern

Database: SQL Server

Auth: Cookie-based (later JWT optional)

UI: Razor Views (Bootstrap)

Single-person team mindset
